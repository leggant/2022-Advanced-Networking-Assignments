Operational Efficiency

Automation reduces the time it takes to deploy a change across our entire network.
For example, A single security patch that requires urgent installation, can be deployed to each device matching the exact make and model the patch is intended for, from a single configuration file, created by a single network administrator. The patch is monitored and can be quickly rolled back to the previous version if necessary.

This one change may have taken days or weeks to test, deploy, and troubleshoot post installation. Automation allows this time critical update to be deployed with no effect to the end-user.

Advantages of Network Automation.

  1. Network wide asset audit: will assess the current network, and provide a list connected assets.
  2. Alerting: will alert network administrators when a device is not in compliance with the standards, has failed to respond to a ping, or has been offline for a period of time.
  3. Network wide configuration audit: will provide a list of devices that are not in compliance with the standards.
  4. Scalability: allow for the deployment of new devices to the network which are compliant with the company requirements.
  5. Backup: will allow for the creation of a backup of the network configuration, and will allow for the restoration of the network configuration from a backup in the event of a natural disaster, outage, or total loss of network.
  6. Software patch management: will allow for the management of software patches, and will allow for a incremental installation of patches on the network. This allows administrators to deploy changes to the network without disruption to services; This is particularly important to our customers and employees different timezones throughout Australia.
